Strings

Strings is an online Music Streaming Service App Design that I made using Figma which could also be used to develop full-fledged app.

Open Figma

Project Overview

Strings App lets user to create an Account and once logged in,The user can explore songs on web player and can follow their favorite artists across the world and play their favorite songs,Create playlist of your choice if not, explore the existing playlist generated based on your music preference.



Strings App currently is just a Design Skin which was designed using popular tool Figma.This application can be run using Figma app On desktop Device or Smartphone for better experience.
